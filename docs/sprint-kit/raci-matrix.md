**Legend:**

- **R** – Responsible (owns execution)
- **A** – Accountable (final decision-making authority)
- **C** – Consulted (provides input before/during work)
- **I** – Informed (kept up to date on progress/outcomes)

> Adapt the matrix per pod as team composition evolves; update runbooks when responsibility shifts.
