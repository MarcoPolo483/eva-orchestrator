# Sprint History Archive

Store completed sprint artifacts here. Each sprint gets its own folder, e.g. `2025-11-sprint-01`, containing:

- Finalized sprint plan
- Ceremony notes (planning, stand-ups, QA gate, review, retro)
- Metrics snapshots and exported dashboards
- Retrospective actions and owners

Use this archive to support audits, future planning, and cross-pod knowledge sharing.
